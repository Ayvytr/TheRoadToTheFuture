# 靠谱：工作基本功

## 沟通技巧

### 先讲结论和PREP模式

```
POINT =结论
REASON =依据
EXAMPLE =具体事例
POINT =重申结论后结束
```

### 直入主题

讲话不要拐弯抹角，要直率、简洁、明确。



* 不找借口，坦率回答是与不是
* 只有明确为什么，才能了解问题所在
* 遇到问题时，提出解决方案
* 有错误直接指出
* 用事实，数据，逻辑说话
* 让对方听得懂，理解对方的期望值
